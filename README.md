# app-adl-students

App for students CRUD in ReactJs

## Getting started

From your terminal run the next commands:

1) Clone repository:
```bash
git clone <GIT REPO HERE>
```
2) Go to the project folder:
```bash
cd <PROJECT FOLDER NAME HERE>
```
3) Install dependencies:

```bash
npm install
```
4) Run the project:
```bash
npm start
```
